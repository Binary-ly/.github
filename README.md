# `.github`

Organization-level configuration and the public profile README for [`github.com/Binary-ly`](https://github.com/Binary-ly).

The contents of [`profile/README.md`](./profile/README.md) render as the org's landing card above the repository list.

## Layout

```
.github/
├── profile/
│   ├── README.md              # rendered at github.com/Binary-ly
│   └── assets/
│       ├── header-dark.png    # 1280×360 hero banner, dark theme
│       ├── header-light.png   # 1280×360 hero banner, light theme
│       ├── header-dark.svg    # source
│       ├── header-light.svg   # source
│       ├── b-mark-dark.svg    # B mark for dark surfaces (cream fill)
│       ├── b-mark-light.svg   # B mark for light surfaces (ink fill)
│       └── social-preview-dark.png  # 1280×640 org social card
└── README.md
```
